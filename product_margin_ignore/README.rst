=======================================
Ignore products from margin calculation
=======================================

* Helper field for margin calculation
* This module does no margin calculation by itself

Configuration
=============
\-

Usage
=====
* Open a product form and check the "Ignore in margin" box
* Install also `sale_order_margin_ignore` and/or 
  `account_invoice_margin_ignore` for the actual calculation and ignore logic
